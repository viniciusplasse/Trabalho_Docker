##Projeto foi desenvolvido na linguagem C# utilizando Framework .NET Core.###

Para executar o "Executável", é necessário primeiro baixar o SDK do .NET core.

Existe disponível para Windows, Linux e MAC OS, conforme link abaixo.

https://dotnet.microsoft.com/download <-  BAIXAR SDK .NET CORE

#Windows#
Após instalar o SDK. Se estiver utilizando sistema operacional WINDOWS basta executar o .exe localizado dentro da pasta do executável.

#Mac/Linux#
Caso esteja no MAC OS ou Linux, após instalar o SDK, abra o console e entre na pasta do Executavel e execute o seguinte comando a seguir:
Comando: dotnet T1_SMA.ddl.

O programa será iniciado via console.

Para utilizar a primeira versão do programa basta informar o número de servidores (Exemplo: 1 ou 2), que o programa fará a simulação conforme o enunciado.


#Projeto#
O código fonte do projeto está localizado na pasta "Projeto", na pasta contém a classe "Program.cs" que consta toda a lógica feita no programa.

Para inicializar o projeto e testar, necessário instalar a editor de texto Visual Code OU visual studio. que tem disponível para Windows e MAC.

Recomendo instalar o Visual studio por ser mais fácil de configurar o ambiente. O visual studio tem a versão free chamado "Visual studio Community".

Visual Code -> https://code.visualstudio.com/download

Visual studio -> https://visualstudio.microsoft.com/pt-br/
